# Linux File System

The Linux file system organizes data into a hierarchical directory structure starting from the root directory (`/`).

Understanding the file system is essential for DevOps engineers because almost every administrative task involves navigating directories, managing files, and configuring system resources.

## Topics Covered

- Linux Directory Structure
- Absolute vs Relative Paths
- File and Directory Commands
- Viewing File Contents
- Searching Files
- Wildcards

## Learning Outcome

After completing this module, I will be able to:

- Navigate Linux directories efficiently
- Create, copy, move, and delete files/directories
- Search for files
- View and edit file contents
- Understand Linux directory hierarchy
